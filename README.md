# OpenClaw 赚钱日记

> 一个人就是一支队伍 - OpenClaw + AI Agent 闭环赚钱系统

## 项目目标

记录 OpenClaw 赚钱之路的每一步：案例学习、技术积累、每日进展、收入复盘。

## 文件夹结构

```
opc/
├── README.md                 # 项目概述（本文件）
├── daily.md                  # 工具资源速查
├── capture.js                # 网页抓取脚本
│
├── 01-案例库/                # 真实赚钱案例收集
│   └── ...                   # 每个案例一个文件
│
├── 02-每日记录/              # 每日赚钱日记
│   └── ...                   # 按日期命名
│
├── 03-技术学习/              # OpenClaw 技术笔记
│   └── ...                   # 入门、配置、最佳实践
│
├── 04-灵感池/                # 待验证的想法
│   └── ideas.md              # 灵感收集
│
├── 05-复盘/                  # 周期性复盘
│   └── ...                   # 周复盘、月复盘
│
└── assets/                   # 图片等资源文件
```

## 核心框架

基于 [opc.md](opc.md) 的一人 OPC 铁闭环赚钱系统：

- **Vibe Coding** - 自然语言编排逻辑，AI Native 编程
- **Agent Memory** - Brain Dump 哲学，长期上下文工程
- **15-Agent 飞轮** - 自动化产出 → 流量测试 → 收入反馈 → 迭代
